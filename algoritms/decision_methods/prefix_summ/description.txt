Префиксные суммы – это алгоритм предварительного вычисления накопительных сумм элементов массива для ускорения запросов на сумму подмассивов.
Принцип работы алгоритма префиксных сумм состоит в создании вспомогательного массива prefix, где prefix[i] содержит сумму первых i элементов исходного массива.
Сумма любого подмассива от индекса l до r вычисляется за O(1) как prefix[r+1] - prefix[l].

Таким образом, алгоритм префиксных сумм имеет следующие показатели сложности:
    Временная сложность: O(n) для построения + O(1) для каждого запроса
    Пространственная сложность: O(n)

Преимущества префиксных сумм:
    Быстрые запросы.
        Сумма любого подмассива вычисляется за постоянное время O(1) после предобработки
    Простота реализации.
        Алгоритм интуитивно понятен и требует минимум кода
    Универсальность.
        Работает для любых запросов на сумму подмассивов любой длины


Недостатки префиксных сумм:
    Дополнительная память.
        Требует O(n) дополнительной памяти для хранения префиксного массива
    Только для сумм.
        Подходит только для задач на суммы, не решает задачи на минимум/максимум

Аналогично списку префиксных сумм можно создать, например, список префиксных произведений и использовать его для быстрого вычисления произведения элементов исходного списка на отрезке [l; r].
